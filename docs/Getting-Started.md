# Getting Started

Please read `Project Overview` first.

## Installing

### Clone the repo

```
git clone $URL
cd $FOLDER
```

### Install dependencies

If you use `yarn`:

```
yarn
```

If you use `npm`:

```
npm install
```

### Building

```
yarn run build
```

### Serving

```
yarn run start
```

### Developing

This will enable autoreload:

```
yarn run dev
```
